## Objective

Perform Environment Setup, Data Cleaning and Exploratory Data Analysis (EDA) on Supermarket Sales Dataset.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

Supermarket Sales Dataset

## Activities Performed

1. Environment Setup using Anaconda
2. Dataset Loading
3. Data Cleaning
4. Missing Value Check
5. Duplicate Removal
6. Exploratory Data Analysis
7. Data Visualization

## Key Insights

* Sales distribution analyzed using histogram
* Product line performance analyzed using bar chart
* Correlation between variables analyzed using heatmap
* Quantity and sales relationship analyzed using scatter plot
